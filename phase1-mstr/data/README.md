# Phase 1 Data Layer

Raw data organized by category.

## Categories

### 01_TRANCHE_HISTORY/
Bitcoin acquisition tranches from strategy.com/purchases

### 02_OBLIGATIONS/
Debt service + preferred dividends from SEC filings

### 03_RESERVE/
USD reserve announcements + history

### 04_BALANCE_SHEET/
Balance sheet extracts (latest 10-Q)

### 05_MARKET_DATA/
Stock prices + BTC prices (daily)

## Data Quality Standards

Tier 1: Verified (±1%)
Tier 2: Well-Sourced (±5%)
Tier 3: Estimated (±10%)
